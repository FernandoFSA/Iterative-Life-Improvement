# Iterative Life Improvement

This repository contains a structured methodology based on the PDCA (Plan–Do–Check–Act) cycle for continuous personal development and self-improvement.

## Structure

- **methodology/**: Core framework and appendices.
- **templates/**: Ready-to-use templates for self-assessment, goal setting, and KPI tracking.
- **examples/**: Sample PDCA cycle in action.
- **changelog/**: Version history and improvements.

## Purpose

To create a systematic, iterative approach to improving life satisfaction and personal performance by applying engineering-style continuous improvement principles.
