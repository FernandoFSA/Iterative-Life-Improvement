# Iterative Life Inprovement

## 1. PLAN (P)

### 1.1 Assess Current State of Life
Evaluate your situation across key life areas (see Appendix A). Identify strengths, weaknesses, and opportunities using structured methods (Appendix B).

### 1.2 Diagnose and Prioritize Problems
1. Describe symptoms using 5W2H.  
2. Formulate and refine hypotheses.  
3. Diagnose issues using MECE and brainstorming tools.  
4. Prioritize based on impact, interconnection, and time demand.

### 1.3 Identify Root Causes
Use tools like “5 Whys” or Ishikawa diagrams to identify underlying causes.

### 1.4 Define and Plan Solutions
1. Brainstorm possible actions.  
2. Evaluate, classify, and select best solutions.  
3. Define goals and deadlines.  
4. Create an execution plan and schedule tasks.

### 1.5 Establish Control Plan
1. Define KPIs and checkpoints.  
2. Set targets and contingency actions.

## 2. DO (D)
Prepare resources, execute planned tasks, and document progress.

## 3. CHECK (C)
Compare actual results to KPIs. Identify deviations and sources of variation.

## 4. ACT (A)
Correct deviations, standardize effective practices, improve methodology, and restart the PDCA cycle.
